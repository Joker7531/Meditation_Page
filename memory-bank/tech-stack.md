## Tech Stack（MVP）

- **Framework**: Next.js (App Router)
- **UI**: React
- **Language**: TypeScript
- **Styling**: Tailwind CSS + PostCSS
- **Animation**: CSS keyframes (`transform`/`opacity`) + `requestAnimationFrame`（timing）
- **State**: React hooks（`useState`/`useReducer`）+ URL query / `sessionStorage`
- **Validation**: Zod
- **Testing**: Vitest + React Testing Library + Playwright
- **Lint/Format**: ESLint + Prettier
- **Deployment**: Vercel
- **Analytics/Monitoring（可选）**: Vercel Analytics / Sentry
